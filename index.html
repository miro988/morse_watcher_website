App Introduction

Three moments that explain how Morse Watcher works.

Tap flashing light

Start by tapping the flashing light source you want to track so Morse Watcher can focus on the correct target. Click + plus icon for higher optical zoom

Automatic calibration starts

After selection, the app automatically calibrates the region of interest and signal settings before decoding.

Morse code is decoded

Decoded Morse symbols, message output, and timing feedback then appear in the live interface.
